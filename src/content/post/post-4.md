- uBlock Origin - 효율적인 차단기.
- Bitwarden - FOS 비밀번호 관리자.
- Violentmonkey - 사용자 스크립트 관리자.
- SmartProxy - 편리한 프록시 관리자입니다. 로컬 WARP-cli 와 함께 사용합니다.
- SingleFile - 모든 페이지 자산을 하나의 파일로 전체 페이지를 저장하는 데 도움이 되는 확장 프로그램입니다.
- Refined GitHub - GitHub 의 사용자 인터페이스와 기능을 향상시키는 확장 기능입니다.
- Raindrop.io - 북마크로 멋진 추억을 만들어 보세요!
- Bing Chat for All Browsers - Edge 외 브라우저에서 Bing AI 를 사용할 수 있습니다.
- Decentraleyes - 콘텐츠 전송 네트워크 (CDN) 의 로컬 에뮬레이션. 온라인 개인 정보 보호 기능을 강화합니다.
- Tabliss - 아름답고 사용자 정의가 가능한 새 탭 페이지.
- Picture-in-Picture Extension
- ModHeader - 몇 번의 클릭만으로 요청 헤더를 변경할 수 있습니다. 저는 이 기능을 사용하여 IP 주소를 스푸핑합니다. X-Forwarded-For 헤더는 Bing AI 에 유용하며 지역 제한이 있습니다.
- LanguageTool - 개인 맞춤법 검사기.
