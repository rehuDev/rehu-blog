# rehu.blog

Astro 기반 개인 블로그 프로젝트.

## Commands

| Command | Description |
| :-- | :-- |
| `npm install` | 의존성 설치 |
| `npm run dev` | 로컬 개발 서버 실행 |
| `npm run build` | 프로덕션 정적 파일 생성 |
| `npm run preview` | 빌드 결과 미리보기 |
| `npm run deploy` | 로컬에서 빌드 후 SSH로 서버 배포 |

## Rule

- 포스팅 제목은 영문 소문자를 사용하며 공백은 하이픈(-)으로 처리한다.
